# Ridgeline EMS — Employee Chat Context

This is a demo deployment of FirePCR. No real employee data is stored here.

## About Ridgeline EMS
Ridgeline EMS is a fictional wildfire medical services company used for demonstration purposes.

## Units
- **Medic 1-4**: Ambulance units (ALS)
- **Aid 1-2**: Medical Support Units
- **Command 1**: Mobile command/supply unit
- **Rescue 1-2**: Technical Rope Rescue teams
